# Brand attribution

Each glyph in `src/brandMarks.tsx` was hand-derived from the brand's
publicly-published official mark / press kit. Where a brand publishes
their official SVG under a permissive license (Anthropic / OpenAI /
Google / Mantle / Aave / MetaMask), the path data here is a simplified
or re-traced approximation suitable for a 24×24 tile — the full official
SVG is not redistributed. Where a brand has no public mark (Cline,
Goose, OpenCode, Codex, etc.), the registry intentionally falls back to
a styled monogram tile.

If your project owns a trademark listed here and you'd like the glyph
removed or replaced, open an issue at
https://github.com/Blockchain-Oracle/mpilot.
