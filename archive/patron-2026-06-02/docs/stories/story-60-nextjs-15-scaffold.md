# Story 60 — Next.js 15 scaffold (apps/web)

**Epic:** Epic 4 — Web App
**Estimated:** ~1.5h
**Depends on:** story-00-monorepo-scaffold, story-02-typescript-config

## BDD Acceptance Criteria

```
Given the monorepo is installed
When `pnpm --filter web dev` runs
Then a Next.js 15 dev server boots on port 3000
And the App Router is active (no /pages directory)
And TypeScript strict mode is enabled (extends tsconfig.base.json)

Given a developer visits http://localhost:3000
When the default page renders
Then it returns 200
And the response body contains "Patron" (placeholder copy is fine)

Given Tailwind v4 is installed
When `pnpm --filter web build` runs
Then the build completes with exit code 0
And the production CSS bundle includes Tailwind v4 utility classes
And no PostCSS warnings appear in the output

Given Biome is configured at the monorepo root
When `pnpm biome check apps/web` runs
Then exit code is 0 on the freshly scaffolded files
And the banned-Tailwind-classes Biome rule (defined per ux-spec) is active
```

## Shell verification

```bash
pnpm install
pnpm --filter web build
test $? -eq 0
pnpm --filter web typecheck
test $? -eq 0
pnpm biome check apps/web
test $? -eq 0

# Dev server boots
pnpm --filter web dev &
DEV_PID=$!
sleep 5
curl -sf http://localhost:3000 | grep -i "patron"
kill $DEV_PID

# App Router (no /pages directory)
test ! -d apps/web/pages
test -d apps/web/app

# Tailwind v4 import present
grep -q '@import "tailwindcss"' apps/web/app/globals.css
```

## Notes

- **Context7 first**: query Next.js 15 + Tailwind v4 docs before writing config. Next 15 App Router and Tailwind v4 PostCSS plugin both changed APIs late 2024.
- React 19 is the Next.js 15 default; do not pin React 18.
- `transpilePackages` is required for workspace imports of `@patron/ui` and `@patron/shared` (they ship raw TSX).
- Do not enable Turbopack production builds yet (still beta in 15.0); dev only.
- Banned Tailwind classes (per ux-spec § "Banned Tailwind classes / patterns") will be enforced via a Biome custom rule in this story OR a follow-up post-install script — add a TODO if not implementable inline.
- Every UI story going forward verifies its slice of the demo-shape rule (per ux-spec).
- File size limit: all files in `apps/web/` enforce the 400-LOC Biome rule from story-01.
